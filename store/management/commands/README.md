## Заполнение базы тестовыми данными

Команда создаёт тестовые данные: пользователей, треки, альбомы, мерч,
корзину, избранное и варианты доставки.

### Использование

```bash
# Создать 20 объектов (значение по умолчанию)
python manage.py seed_db

# Создать 100 объектов
python manage.py seed_db 100
